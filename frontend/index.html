¿Cómo usar el sistema?

1

Gestiona Productos

Agrega, edita y elimina productos de tu inventario. Controla automáticamente el stock de cada producto.

2

Registra Clientes

Crea una base de datos de clientes con su información de contacto y tipos de identificación.

3

Realiza Ventas

Registra ventas vinculando productos con clientes. El sistema calcula automáticamente totales.

4

Consulta Reportes

Visualiza estadísticas en tiempo real de tus productos, clientes y ventas.
